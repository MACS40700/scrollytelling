Interactive Charts with ggiraph

Getting Started with ggiraph

ggiraph extends ggplot2 with interactivity — hover tooltips, highlights, and click actions — using the same grammar you already know. As you scroll, each chart will appear alongside explanations.

Box Plots

Box plots are great for comparing distributions across groups. Here we’re looking at the percentage of college-educated residents across Midwestern states. Hover over each box to see the state name and value. Notice how Illinois has a wider spread compared to Indiana.

Bar Charts

Bar charts are ideal for categorical comparisons. With ggiraph, swap geom_col() for geom_col_interactive() and add a tooltip aesthetic — that’s all it takes. Hover over each bar to see the value.

Scatter Plots

Scatter plots reveal relationships between two continuous variables. The iris dataset shows a clear positive correlation between sepal length and petal length. Hover over any point to see the exact measurements. The three species separate naturally in space.

Dot Plots

With ggiraph, the pattern is always the same: take any geom_*() from ggplot2, add _interactive, and pass tooltip and data_id aesthetics. Hover over the dots to see each car’s MPG and cylinder count.

Dumbbell Plots

Dumbbell plots excel at showing change between two time points. Each line connects 1952 (purple) to 2007 (yellow) for each Asian country. Hover over any point to see the exact life expectancy. The dramatic rightward shift represents one of the greatest improvements in human longevity in recorded history.
