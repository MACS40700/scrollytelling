Interactive Plotly Charts

Getting Started with Plotly

Plotly makes it easy to create beautiful interactive visualizations in R. As you scroll, each chart will appear in sequence alongside explanations.

Box Plots

Box plots are great for comparing distributions across groups. Here we’re looking at the percentage of college-educated residents across Midwestern states. Notice how Illinois and Michigan have wider spreads compared to Indiana.

Bar Charts

Bar charts are ideal for categorical comparisons. The type = "bar" argument is the key — plotly supports many chart types (scatter, box, histogram, heatmap) using the same syntax.

Scatter Plots

Scatter plots reveal relationships between two continuous variables. The iris dataset shows a clear positive correlation between sepal length and petal length — and the species separate naturally in space.

The ggplotly() function wraps any ggplot2 object into an interactive plot. This is the fastest route to interactivity — build your chart in ggplot2, then simply pass it to ggplotly(). Hover over the dots to see exact values!

Dumbbell Plots

Dumbbell plots excel at showing change between two time points. Each line connects 1952 (purple) to 2007 (yellow) for each Asian country. The dramatic rightward shift represents one of the greatest improvements in human longevity in recorded history.
